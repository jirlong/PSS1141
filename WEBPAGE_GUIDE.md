# AI 智能助手平台 - 網頁使用指南

## 概述

這是一個現代化的 AI 服務網頁介面，提供五大智能服務：
- 💬 一般問答
- 🌐 智能翻譯
- 📰 新聞分析
- 📖 故事創作
- 🎯 智能分派

## 快速開始

### 1. 啟動 Flask 應用

```bash
python flask_app.py
```

應用會在 `http://localhost:5001` 上運行。

### 2. 訪問網頁

在瀏覽器中打開：`http://localhost:5001`

## 功能介紹

### 💬 一般問答服務
- 向 AI 提出任何問題
- 支援多種 AI 模型選擇
- 適用於知識查詢、解釋說明等

**使用範例：**
- "什麼是人工智慧？"
- "請解釋量子計算的原理"
- "如何學習 Python？"

### 🌐 智能翻譯服務
- 自動識別中文/英文
- 雙向智能翻譯
- 保留原文對照

**使用範例：**
- 輸入中文自動翻譯成英文
- 輸入英文自動翻譯成中文

### 📰 新聞 5W1H 分析
- 自動提取新聞關鍵資訊
- 結構化呈現 5W1H（誰、什麼、何時、何地、為何、如何）
- 適用於新聞摘要、快速理解

**使用方式：**
1. 複製新聞內容
2. 貼到輸入框
3. 點擊"分析新聞"
4. 查看結構化分析結果

### 📖 AI 故事創作
- 根據主題創作故事
- 可自訂字數長度
- 支援多種寫作風格：
  - 村上春樹風格
  - 童話風格
  - 冒險風格
  - 詩意風格

**使用範例：**
- 主題：友誼
- 字數：100-500
- 風格：選擇喜歡的風格

### 🎯 智能任務分派
- 自動判斷使用者意圖
- 智能路由到對應服務
- 一站式處理各種需求

**使用範例：**
- "請幫我翻譯 Hello World"（自動識別為翻譯任務）
- "寫一個關於友誼的故事"（自動識別為故事創作）
- "什麼是機器學習？"（自動識別為問答任務）

## 技術架構

```
Frontend:
├── HTML5 (templates/index.html)
├── CSS3 (static/style.css)
└── JavaScript (static/script.js)

Backend:
└── Flask (flask_app.py)
    ├── API 端點
    ├── OpenAI 整合
    └── JSON 回應處理
```

## API 端點

所有功能都透過以下 API 端點實現：

- `POST /api/ask` - 一般問答
- `POST /api/translate` - 翻譯服務
- `POST /api/news-summary` - 新聞分析
- `POST /api/create-story` - 故事創作
- `POST /api/dispatch` - 智能分派

## 故障排除

### 問題：頁面無法載入
**解決方案：**
1. 確認 Flask 應用正在運行
2. 檢查 port 5001 是否被佔用
3. 查看終端錯誤訊息

### 問題：API 請求失敗
**解決方案：**
1. 確認 OpenAI API Key 設定正確
2. 檢查網路連接
3. 查看瀏覽器控制台錯誤

### 問題：樣式顯示異常
**解決方案：**
1. 清除瀏覽器快取
2. 硬性重新整理（Ctrl+Shift+R 或 Cmd+Shift+R）
3. 確認 static 資料夾路徑正確
